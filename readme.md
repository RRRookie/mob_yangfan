	（Baidu Institute of Front-End Technology即百度Web前端技术学院）

我们的队伍名称叫做Big Hero 2 
	（超能陆战队）

我们选择的班级是移动扬帆班

	面向群体：有一定编程经验的同学

	学习任务一：移动记账簿

	实现一个在移动设备浏览器上使用的简单的记账簿，可以实现简单的增加账目，查看账目的功能

	学习任务二：移动高级记账簿

	在任务一基础上，加强体验，支持统计功能，增加图表等

我们的目标是

	一、将所学知识与实践快速结合起来，积累项目经验

	二、通过团队协作，互帮互助，互相督促，共同成长
